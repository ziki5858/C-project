\doxysubsection{Detailed Description}
This file contains the implementation of the functions that used to create the structs of the assembler. the file contains the following functions\+:

\item create\+\_\+constant -\/ a function that creates a constant.
\item create\+\_\+external -\/ a function that creates an external.
